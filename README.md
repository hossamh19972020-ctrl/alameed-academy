# أكاديمية العميد التعليمية — Android Build Ready

هذه النسخة تحتوي على مشروع Flutter كامل مع ملفات Android وإعداد GitHub Actions لبناء ملف APK تلقائيًا.

## بيانات المشروع

- اسم التطبيق: أكاديمية العميد التعليمية
- Package ID: `com.alameed.academy`
- إصدار التطبيق: `0.3.0+3`
- حساب GitHub المقترح: `hossamh19972020`
- بريد التواصل: `hossamh19972020@gmail.com`

## بناء APK من الهاتف عبر GitHub

1. أنشئ مستودعًا جديدًا على GitHub باسم `alameed-academy`.
2. ارفع **محتويات هذا المجلد** إلى المستودع، وليس المجلد الخارجي فقط.
3. افتح تبويب **Actions**.
4. اختر **Build Android APK**.
5. اضغط **Run workflow** ثم انتظر اكتمال البناء.
6. افتح نتيجة البناء، ثم نزّل الملف من قسم **Artifacts** باسم `alameed-academy-apk`.
7. فك الضغط وستجد `app-release.apk`، ثم ثبّته على هاتف Android.
